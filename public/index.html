QABot — Mock NGFT/Axon Batch Service

Simulates steps 4-5-6 of the back-office batch pipeline for QA test generation.

Endpoints

POST /upload — Upload ISO file to mock NGFT
Alias: /.netlify/functions/upload
GET /batch/:fileId/status — Check Axon batch job status
Alias: /.netlify/functions/batchStatus/:fileId
GET /reconcile/:fileId — Get reconciliation result
Alias: /.netlify/functions/reconcile/:fileId

Contracts

  • Upload Contract (JSON)
  • Batch State Transitions (JSON)
  • Reconciliation Rules (Markdown)

Quick Test (cURL)

# 1. Upload file → QUEUED
curl -X POST https://YOUR_NETLIFY_URL/.netlify/functions/upload \
  -H "Content-Type: application/json" \
  -d '{"filename":"test.xml","contentHash":"hash001","isoType":"ISO20022"}'

# 2. Check batch status → PROCESSED
curl https://YOUR_NETLIFY_URL/.netlify/functions/batchStatus/NGFT-1001-testxml0

# 3. Get reconciliation → RECONCILED
curl https://YOUR_NETLIFY_URL/.netlify/functions/reconcile/NGFT-1001-testxml0

# 4. Get reconciliation → EXCEPTION
curl https://YOUR_NETLIFY_URL/.netlify/functions/reconcile/NGFT-1001-testxml1

6-Step Business Flow

  1. Business partner create — API hit and store data (10–15) — UI (Selenium)
  2. Insert data into 12 tables
  3. ISO format file — input file
  4. Upload to NGFT → POST /upload
  5. Axon picks + batch job → GET /batch/:id/status
  6. Confirm exception or reconciled → GET /reconcile/:id
